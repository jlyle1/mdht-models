  1. SHALL contain exactly one [1..1] code (CONF:51)/@code="LOINC_TBD_001" External Document Section (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:50)
  2. SHALL contain exactly one [1..1] text (CONF:52)
  3. SHALL contain exactly one [1..1] title and SHOULD equal "External Document" (CONF:53)
  4. SHALL contain at least one [1..*] entry {subsets Section::entry} (CONF:55)
    1. Such entries SHALL contain exactly one [1..1] outcomesPriorDocument
      1. This outcomesPriorDocument SHALL contain exactly one [1..1] code, which SHALL be selected from ValueSet STATIC (CONF:25)

        NEMSIS Trace:  eOutcome.03 - External Report ID/Number Type, eOutcome.05 - Other Report Registry Type. If the type is not found, the type code may be null and the other type text placed in the code original text attribute.

      2. This outcomesPriorDocument SHALL contain exactly one [1..1] id (CONF:26)

        NEMSIS Trace: eOutcome.04 - External Report ID/Number. This will typically be placed in the extension, as NEMSIS will not maintain an OID registry for all responder and healthcare organizations. The vendor or implementer may discover or assign OIDS.
